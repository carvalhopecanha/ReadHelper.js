# ReadHelper.js

ReadHelper.js is a lightweight JavaScript script for **guided reading**, automatically highlighting text to help with **focus, accessibility, and learning**.

This project is experimental and still under development, but already works as a proof of concept.

---

ReadHelper.js é um script leve em JavaScript para **leitura guiada**, que destaca automaticamente o texto para auxiliar no **foco, acessibilidade e aprendizado**.

O projeto ainda está em desenvolvimento, mas já funciona como prova de conceito.
